01 — What we do

Professional, quality
electrical work.

Four things, done properly. If your home needs it wired, fixed, replaced or built out, it lands in one of these.

Repairs

Dead outlets, tripping breakers, flickering lights, panels that hum. We find the actual cause instead of guessing at it.

Installations

Fixtures, fans, outlets, switches, dedicated circuits. New work installed to code and finished clean.

Build outs

Additions, remodels, garages and finished-out spaces wired from the rough-in through the final trim.
